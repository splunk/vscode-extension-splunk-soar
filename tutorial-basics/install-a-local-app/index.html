Install a Local App

There are multiple ways to get a local app installed on a configured environment.

  • with a VS Code Task (recommended)
  • using the Editor bar button
  • with the File Explorer context menu
  • Packaged upload via the Apps view

Which one works best for you depends on your usual workflow. We recommend the VS Code Task workflow as it can be used with keybindings and offers the best developer experience after initial setup.
